# Node.js版 ソースコード

GCP Cloud Run functions で動作するAIチャットアプリのNode.js版ソースコードです。

## ファイル一覧

| ファイル | 役割 | GCPでの操作 |
|---------|------|------------|
| [backend.js](backend.md) | サーバー側の処理（メイン） | index.js に貼り付け |
| [front.html](front.md) | 画面のデザイン（HTML/CSS） | 新規ファイルとして追加 |
| [script.js](script.md) | フロントエンドの処理（JavaScript） | 新規ファイルとして追加 |
| [package.json](package.md) | 使用するライブラリ一覧 | package.json に貼り付け |

## 使い方

1. 各ファイルをクリックしてコードを表示
2. コードブロック右上のコピーボタンでコピー
3. GCPのCloud Run functionsに貼り付け

詳しい手順は [構築手順書](../../gcp-firebase/02_setup.md) を参照してください。
